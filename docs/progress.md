# Dengbej AI Development Progress

This document tracks the development progress of the Dengbej AI project during the build phase of the AWS 10,000 AIdeas Competition.

Dengbej AI explores how artificial intelligence can transform written articles into narrated storytelling experiences inspired by the Kurdish dengbêj oral tradition.

---

## Project Idea

The goal of Dengbej AI is to build a simple web platform where users can submit written content and receive:

- A summarized version of the text generated by AI
- Optional translation into Kurdish
- A narrated audio version of the story

The project combines modern AI services with serverless cloud architecture to create accessible storytelling experiences.

## Phase 2 — Frontend Prototype

A basic frontend interface was created to allow users to interact with the system.

Features implemented:

- Text input field for pasting article content
- Button to trigger AI story generation
- Summary display area
- Audio player for narrated output
- Connection to backend via AWS Lambda Function URL

The frontend is implemented using simple HTML and JavaScript.

---

## Phase 3 — Backend Integration

The backend processing pipeline was implemented using AWS serverless services.

Implemented components:

- AWS Lambda function for backend orchestration
- Amazon Bedrock for AI summarization
- Amazon Polly for speech synthesis
- Amazon S3 for storing generated audio files

Lambda receives the user request, processes the content using AI services, generates the narration, stores the audio in S3, and returns a playable audio URL to the frontend.
