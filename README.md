# Dengbej AI

Dengbej AI is an experimental storytelling platform inspired by the Kurdish dengbêj oral tradition. The project explores how AI can transform written articles into short narrated audio stories.

## Prototype Features
- AI-powered article summarization
- Optional multilingual translation
- Audio narration generation
- Serverless cloud architecture

## Architecture
The prototype uses AWS services:

- Amazon Bedrock – AI text generation
- AWS Lambda – serverless processing
- Amazon Polly – text-to-speech narration
- Amazon S3 – storage for generated audio

## Project Goal
This prototype is being developed for the AWS 10,000 AIdeas Competition and explores how modern AI tools can support storytelling and accessible media.
